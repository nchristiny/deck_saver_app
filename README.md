# DECK SAVER 
(under construction)
### By Nicholas Christiny  
Deck Saver is a web app using the public [Hearthstone API](http://hearthstoneapi.com/ "Hearthstone API") to save various constructed decks for each class.
### Dependencies:
	- npm
	- React
	- backend database connection
	- valid API key
### Usage:
```
npm install
npm run dev
```

http://localhost:9000/
